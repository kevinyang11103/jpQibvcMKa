# 项目简介 python689

本项目是一款基于 Python 和 Flask 开发的小型论坛系统，采用前后端分离的模式，旨在为用户提供一个简洁、高效的信息交流平台。

## 技术栈

- 前端：HTML/CSS/JavaScript
- 后端：Python + Flask
- 数据库：MySQL

## 功能模块

### 用户模块

- 登陆和注册
- 发表帖子
- 发表评论

### 管理员模块

- 管理用户
- 管理评论
- 管理帖子

## 系统角色

- 用户：普通用户，可进行登录、注册、发帖、评论等操作。
- 管理员：负责管理用户、评论和帖子，维护论坛秩序。

## 运行环境

- Python 3.6+
- Flask 1.0+
- MySQL 5.7+

## 部署步骤

1. 安装 Python、Flask 和 MySQL 环境
2. 创建数据库和表结构
3. 克隆项目代码到本地
4. 配置项目环境
5. 运行项目
